### 3.1 Low Adoption Playbook

**Trigger**: MAU <40% after 30 days

**Root Cause Analysis:**
```
â–¡ Training gap - Users don't know how to use
â–¡ Value gap - Users don't see the benefit
â–¡ Access gap - Users can't access easily
â–¡ Champion gap - No internal advocate
â–¡ Use case gap - Wrong use cases identified
```

**Action Plan:**

| Week | Actions | Owner |
|------|---------|-------|
| 1 | Call with champion to understand barriers | CSM |
| 1 | Review usage data by user | CSM |
| 2 | Offer targeted training for low-usage users | CSM |
| 2 | Identify and empower super users | CSM |
| 3 | Executive check-in if no improvement | CSM + Manager |
| 4 | Escalate to renewal risk if <50% MAU | CSM + Manager |

## Part 5: Expansion Strategies

### 5.1 Expansion Opportunity Types

| Type | Description | Trigger |
|------|-------------|---------|
| **User Expansion** | More users in same team | High adoption, waitlist |
| **Team Expansion** | New departments/teams | Cross-functional interest |
| **Use Case Expansion** | New use cases | Success with primary use case |
| **Feature Expansion** | Higher tier, new modules | Feature requests, growth |

### 5.2 Expansion Discovery Questions

**User Expansion:**
```
- Are there other team members who would benefit from VITAL?
- Are there people waiting to get access?
- Any new hires who should be added?
```

**Team Expansion:**
```
- Which other teams have similar challenges?
- Has anyone from other departments asked about VITAL?
- Are there cross-functional projects where VITAL could help?
```

**Use Case Expansion:**
```
- What other workflows could benefit from AI expertise?
- Are there pain points we haven't addressed yet?
- What are your priorities for next quarter?
```

**Feature Expansion:**
```
- Have you tried [Mode 3/4] for complex tasks?
- Would multi-expert panels (Ask Panel) be valuable?
- Are there enterprise features you need (SSO, audit)?
```

### 6.3 Save Playbook

**Trigger**: Customer indicates cancellation intent

**Immediate Actions (24 hours):**
```
â–¡ Acknowledge and express concern
â–¡ Schedule emergency call
â–¡ Loop in CSM manager
â–¡ Review all account history
â–¡ Prepare retention offer options
```

**Discovery Questions:**
```
- What's driving this decision?
- Is this final, or can we discuss?
- What would need to change for you to stay?
- What did we do well? What fell short?
- Is there a competitor involved?
```

**Save Offer Options:**

| Offer Type | When to Use | Approval |
|------------|-------------|----------|
| Extended pilot | Value not proven | CSM Manager |
| Discount | Price objection | Sales Manager |
| Additional training | Adoption issue | CSM |
| Feature commitment | Missing capability | Product + VP |
| Contract flexibility | Term concerns | Legal + VP |

**Save Conversation Script:**
```
"I really appreciate you sharing this with me. Before we proceed
with cancellation, I'd love to understand what's behind this
decision so we can either address it or learn from it.

[Listen carefully, take notes]

Based on what you've shared, here's what I can offer:
[Present tailored save offer]

Would this address your concerns? I want to make sure we've
explored every option before we part ways."
```
