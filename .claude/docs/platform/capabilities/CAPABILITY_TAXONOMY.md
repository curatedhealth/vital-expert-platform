## Structure

Each capability includes:
- **ID**: Unique identifier (e.g., CAP-MA-001)
- **Name**: Capability name
- **Definition**: Clear description of what the capability entails
- **Level Scope**: Which agent levels this applies to (L1/L2/L3/L4/L5)
- **Function/Role Context**: Where this is typically used
- **Related Skills**: Child skills under this capability
