## Archetype Cheat Sheet

### AUTOMATOR
- **Profile**: High AI + Routine work
- **Persona**: "Automate everything possible"
- **Top Pains**: Manual processes (1.8x), Tech limitations (1.6x), Time waste (1.5x)
- **Solution Fit**: Automation, workflows, AI tools
- **Communication**: Show time savings, efficiency metrics
- **Examples**: PP-PROC-001 (data entry), PP-PROC-008 (literature search)

### ORCHESTRATOR
- **Profile**: High AI + Strategic work
- **Persona**: "Give me insights, not just data"
- **Top Pains**: Knowledge gaps (1.8x), Tech insights (1.7x), Org barriers (1.7x)
- **Solution Fit**: Insights, analytics, intelligence platforms
- **Communication**: Show strategic value, competitive edge
- **Examples**: PP-TECH-002 (integration), PP-TECH-005 (AI insights)

### LEARNER
- **Profile**: Low AI + Routine work
- **Persona**: "Make it simple and teach me"
- **Top Pains**: Knowledge gaps (2.0x), Tech complexity (1.9x), Compliance fear (1.6x)
- **Solution Fit**: Training, guided workflows, simple interfaces
- **Communication**: Emphasize simplicity, provide support
- **Examples**: PP-KNOW-004 (learning curve), PP-TECH-001 (CRM complexity)

### SKEPTIC
- **Profile**: Low AI + Strategic work
- **Persona**: "Prove it works first"
- **Top Pains**: Unproven tech (2.0x), Org change (1.9x), Compliance risk (1.8x)
- **Solution Fit**: Proven tech, evidence-based, incremental
- **Communication**: Case studies, pilot programs, risk mitigation
- **Examples**: PP-TECH-006 (downtime), PP-COMP-003 (violation fear)

## Usage Checklists

### For Product Discovery
- [ ] Identify target role(s)
- [ ] Assess archetype distribution
- [ ] Review top 5 pain points for role
- [ ] Check archetype weights for pains
- [ ] Calculate VPANES scores
- [ ] Map to existing opportunities
- [ ] Prioritize by: shared + VPANES + solvability

### For Sales/Demo
- [ ] Ask role and experience level
- [ ] Probe for archetype indicators
- [ ] Lead with top 3 role pains
- [ ] Show before/after workflows
- [ ] Quantify time/quality impact
- [ ] Address archetype objections
- [ ] Connect to opportunity solution

### For Feature Prioritization
- [ ] Count affected roles (shared pain?)
- [ ] Sum VPANES across archetypes
- [ ] Assess solvability (quick win?)
- [ ] Check opportunity coverage
- [ ] Estimate implementation effort
- [ ] Calculate ROI (impact / effort)
- [ ] Validate with user research
