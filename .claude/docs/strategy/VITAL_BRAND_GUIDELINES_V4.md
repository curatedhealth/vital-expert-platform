## 3. AgentOS 3.0 Hierarchy {#agentos-hierarchy}

AgentOS 3.0 is the intelligence operating system of VITAL. It orchestrates 300+ domain experts, specialists, and tools. We visualize this hierarchy through specific geometric rules.

### Level 1: MASTER (Orchestrators)

- **Role**: Strategic conductors coordinating multiple experts end-to-end.
- **Visual**: "Super Orchestrator" Icon. Halo + Central Hub + Connected Nodes.
- **Usage**: High-stakes, multi-domain orchestration (Mode 3).

### Level 2: EXPERT (Domain Specialists)

- **Role**: The primary user-facing consultants (e.g., "Regulatory Strategy Director").
- **Visual**: Geometric Avatars. Circle head + Abstract geometric torso.
- **Color**: Strong Tenant Color (Purple, Blue, Black, etc.).
- **Usage**: The primary cards in the Agent Grid.

### Level 3: SPECIALIST (Sub-Experts)

- **Role**: Focused sub-domain capabilities (e.g., "Labeling Analyst") spawned by Experts.
- **Visual**: Smaller, angular forms (Diamonds/Triangles). No halos.
- **Color**: Desaturated/Lighter version of parent Tenant Color.
- **Usage**: Nested in trace views or panels.

### Level 4: WORKER (Task Executors)

- **Role**: Invisible operations team executing structured tasks (formatting, scraping).
- **Visual**: Minimal monochrome shapes (Rectangles, Bars).
- **Color**: Neutral Grays.
- **Usage**: Only visible in logs or execution traces.

### Level 5: TOOL (Integrations)

- **Role**: Precision instruments and APIs (e.g., PubMed Search, Calculator).
- **Visual**: Pure geometry (Hexagons, Gears, Plugs). No "head" shape.
- **Color**: Monochrome Neutral.
- **Usage**: Listed in Tools Library or execution logs.

## 6. Iconography System {#iconography}

VITAL's icons are geometric, modernist, and abstract. They look like siblings of the Avatar system.

### 6.1 Icon Grammar

- **Construction**: Built strictly from Atomic Shapes (Circle, Square, Triangle, Line).
- **Style**: 2px Stroke, Minimal Fills.
- **No Metaphors**: No "magnifying glasses" for search unless abstractly constructed.
- **Color**: Neutral (`#1A1A1A`) stroke. Accent fill on ONE atomic shape for meaning.

### 6.2 Dashboard Icon Concept (The "Grid & Insight" Family)

- **2x2 Grid**: Represents Modules/Data.
- **Radiant Node**: Represents Intelligence/Insight.
- **Combination**: A circle (insight) hovering over a grid (data) represents the VITAL Dashboard.

## 8. UI & Layout {#ui-layout}

### 8.1 Philosophy

**"Warm Modernism"**

- **Backgrounds**: Always `#FAF8F1` (Warm Ivory), never stark white (`#FFFFFF`) for main pages.
- **Cards**: Soft rounded corners (8px radius), subtle borders (Neutral-200), minimal shadows.
- **Spacing**: Generous. 8pt grid system.

### 8.2 Agent Cards

- **Header**: Agent Role (Bold).
- **Visual**: Large Geometric Avatar (Left or Center).
- **Tags**: Tier, Function, Tenant Color.
- **Footer**: Actions (Chat, Configure).
