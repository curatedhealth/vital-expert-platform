# VITAL Ask Expert - User Guide

## 🎯 Overview

VITAL Ask Expert is an advanced AI-powered consultation platform that connects you with specialized healthcare AI agents. Choose from 50+ expert agents, select the tools they can use, and watch their reasoning process in real-time.

## 🚀 Getting Started

### 1. Accessing Ask Expert
- Navigate to `/ask-expert` in your VITAL platform
- You'll see the main interface with agent selection on the left and chat area on the right

### 2. Understanding the Interface
- **Left Sidebar**: Agent selection and tool configuration
- **Main Chat Area**: Conversation with selected expert
- **Reasoning Display**: Real-time AI thinking process (when active)

## 👨‍⚕️ Selecting an Expert Agent

### Manual Selection Mode (Recommended)
1. **Browse Agents**: Scroll through the list of available experts
2. **Search**: Use the search bar to find specific specialties
3. **Filter by Tier**: 
   - **Tier 1**: General healthcare knowledge
   - **Tier 2**: Specialized expertise
   - **Tier 3**: Advanced specialist knowledge
4. **Select Agent**: Click on an agent card to select them

### Automatic Selection Mode
1. **Type Your Question**: Enter your medical question
2. **Send Message**: The system will analyze your question
3. **Choose from Suggestions**: Select from recommended experts
4. **Continue Conversation**: Chat with the selected expert

## 🛠️ Tool Selection

### Available Tool Categories

#### Research Tools
- **Web Search**: Current internet information
- **PubMed Search**: Medical literature database
- **Cochrane Reviews**: Evidence-based medical reviews

#### Regulatory Tools
- **FDA Database**: Drug approvals and safety data
- **EMA Database**: European Medicines Agency data
- **Clinical Trials**: Clinical study information

#### Analysis Tools
- **Drug Interaction Checker**: Medication interaction analysis
- **Medical Calculator**: Clinical calculations
- **RAG Search**: Internal knowledge base search

### Selecting Tools
1. **Choose Tools**: Check the tools you want the agent to use
2. **Tool Categories**: Tools are organized by category for easy selection
3. **Agent Compatibility**: Only compatible tools are shown for each agent
4. **Real-time Updates**: Tool selection updates immediately

## 💬 Having a Conversation

### Sending Messages
1. **Type Your Question**: Use the message input at the bottom
2. **Be Specific**: Provide relevant details about your condition or question
3. **Send Message**: Click the send button or press Enter
4. **Wait for Response**: The agent will process your question using selected tools

### Understanding Responses
- **Expert Analysis**: Detailed response from your selected expert
- **Source Citations**: References to medical literature and databases
- **Tool Usage**: See which tools were used in the analysis
- **Confidence Levels**: Understanding of the agent's confidence in the response

## 🧠 Understanding AI Reasoning

### Real-time Reasoning Display
When an agent is processing your question, you'll see:
1. **Step-by-step Process**: Each step of the AI's analysis
2. **Tool Usage**: Which tools are being used and when
3. **Data Sources**: What information is being gathered
4. **Synthesis**: How the information is being combined

### Reasoning Steps
- **Analysis**: Understanding your question
- **Research**: Gathering relevant information
- **Synthesis**: Combining findings into a response
- **Validation**: Checking the response for accuracy

## 📊 Best Practices

### Getting Better Results
1. **Be Specific**: Provide detailed information about your question
2. **Include Context**: Mention relevant medical history or current medications
3. **Ask Follow-ups**: Don't hesitate to ask clarifying questions
4. **Use Multiple Tools**: Select relevant tools for comprehensive analysis

### Question Examples
**Good Questions:**
- "What are the side effects of metformin for a 65-year-old with type 2 diabetes?"
- "What are the latest treatment options for rheumatoid arthritis?"
- "How do I manage hypertension with a low-sodium diet?"

**Avoid:**
- "Am I sick?" (too vague)
- "What's wrong with me?" (needs more context)
- "Should I take this medication?" (requires medical consultation)

## 🔧 Troubleshooting

### Common Issues

#### Agent Not Responding
- **Check Selection**: Ensure an agent is selected
- **Verify Tools**: Make sure at least one tool is selected
- **Try Again**: Send the message again
- **Switch Agents**: Try a different expert

#### Slow Responses
- **Tool Selection**: Too many tools can slow down responses
- **Question Complexity**: Complex questions take longer to process
- **System Load**: High usage may cause delays

#### Error Messages
- **Validation Errors**: Check your message format
- **Tool Errors**: Some tools may be temporarily unavailable
- **Network Issues**: Check your internet connection

### Getting Help
- **Contact Support**: Use the help button in the interface
- **Check Status**: Look for system status indicators
- **Try Different Agent**: Switch to a different expert
- **Simplify Question**: Break complex questions into smaller parts
